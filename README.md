# ByteRight

ByteRight is a student-focused meal planning web application designed to help users eat better on a budget. It combines personalised meal planning, automatic shopping list generation, recipe discovery, and a simple social feed into one platform.

---

## Features

* User registration and login
* Personalised weekly meal plan generation
* Automatic shopping list based on selected meals
* Recipe browsing, filtering, and saving
* Trending and budget-friendly recipe recommendations
* Social feed with posts, likes, and comments

---

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Environment:** XAMPP / Apache

---

## Project Structure

* `frontend/` – user interface (HTML, CSS, JavaScript)
* `backend/` – API endpoints, database configuration, setup script, and schema
* `README.md` – project documentation

---

## Setup Instructions

1. Place the `Byte-Right` folder inside your XAMPP `htdocs` directory.

2. Start **Apache** and **MySQL** using the XAMPP control panel.

3. Run the setup script in your browser:

   http://localhost/Byte-Right/backend/setup.php

4. Open the application:

   http://localhost/Byte-Right/frontend/byteright_login.html
