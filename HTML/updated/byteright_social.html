Data Flow

System Overview

Login → Home

User logs in, session starts, homepage loads personalized content.

Home → Features

Home links to recipe generator, planner, templates, etc., checking session each time.

Pages ↔ Database

Each feature page sends requests to PHP, which reads/writes MySQL data.

Saving Data

Forms send POST requests, PHP updates the database, and pages refresh with new data.

Consistent Navigation

Same navbar across pages so users move smoothly through the site.
